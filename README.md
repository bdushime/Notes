# Note Taking Web App

A clean, responsive, DOM-based Note Taking Web App built with plain HTML, CSS, and Vanilla JavaScript.

## 🔗 Live Demo
[View the Live Web App Here](https://noteslll.netlify.app/)

##  Features
* **Authentication**: Simulated user login and session handling using the Web Storage API (`localStorage` & `sessionStorage`).
* **Note Management**: Create, view, edit, search, and delete notes dynamically.
* **Tags System**: Categorize and filter notes by custom tags.
* **Theming**: Dark and light mode toggle with customization options.
* **Responsive Design**: Mobile-friendly sidebars and settings menus.

##  Built With
* Vanilla JavaScript (ES6+)
* HTML5 & CSS3
* Custom DOM Rendering Engine
* Web Storage API

##  Local Execution
Simply clone this repository and open `index.html` in your favorite web browser to run it locally. No build step required!
